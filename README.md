# Anti-Money-Laundering

🚨 Built an Agentic AI System for Anti-Money Laundering (AML) using OpenAI Agent Builder

1) Financial crime is no longer rule-based.

2) It’s behavioural. It’s adaptive. And now it’s AI-driven.

3) So I built an AI-powered Anti Money Laundering (AML) Agent that simulates how modern financial institutions detect suspicious transactions using intelligent, multi-agent orchestration.

4) The objective was to design an Agentic AI workflow capable of analysing transactions, scoring risk, and generating structured compliance reports.

Here’s what I implemented 👇


🧠 AI Architecture (Agentic Design)

This is NOT a single prompt system.
It is a multi-agent architecture built using OpenAI Agent Builder:

1️⃣ Transaction Analyser Agent - 

(a) Performs contextual transaction evaluation

(b) Detects anomalies in transaction patterns

(c) Applies behavioural heuristics

(d) Extracts structured financial intelligence


2️⃣ Risk Scorer Agent

(a) Applies dynamic risk modelling

(b) Assigns LOW / MEDIUM / HIGH classification

(c) Simulates probabilistic risk assessment

(d) Uses classification logic (threshold-based scoring)


3️⃣ Decision Engine (If/Else Logic Layer)

(a) Conditional routing based on risk output

(b) Demonstrates deterministic + probabilistic AI flow control


4️⃣ Compliance Report Generator Agent

(a) Converts analysis into structured compliance documentation

(b) SAR-style reporting simulation

(c) Natural Language Generation (NLG) for regulatory use cases
